# Análise Socioeconômica — Dados IBGE

Análise exploratória de dados socioeconômicos da população brasileira, com foco em renda, escolaridade, cor/raça e distribuição regional.

---

##  Sobre o Dataset

| Info | Valor |
|---|---|
| Fonte | IBGE (Instituto Brasileiro de Geografia e Estatística) |
| Total de registros | 76.840 pessoas |
| Variáveis | UF, Sexo, Idade, Cor, Anos de Estudo, Renda, Altura |

---

##  Análises Realizadas

###  Análises Gerais
- Renda média geral da população
- Idade média da população
- Distribuição por estado (UF)
- Distribuição por cor/raça

### ‍ Análises de Gênero
- Diferença de renda média entre homens e mulheres
- Anos de estudo por sexo
- Distribuição de altura por sexo

###  Análises de Renda
- Distribuição por classe social (A, B, C, D, E)
- Estado mais rico e mais pobre
- Cor/raça com maior e menor renda média

###  Análises de Educação
- Anos de estudo por cor/raça
- Anos de estudo por estado
- Relação entre escolaridade e renda

---

##  Principais Descobertas

### Renda
- A renda média geral é de **R$ 2.000,38**
- Homens ganham em média **R$ 2.192,44** vs **R$ 1.566,85** das mulheres — diferença de **28,5%**
- O **Distrito Federal** é o estado com maior renda média
- O **Maranhão** é o estado com menor renda média

### Classe Social
| Classe | Faixa de Renda | % da População |
|---|---|---|
| E | Até R$ 1.576 | 64,8% |
| D | Até R$ 3.152 | 21,7% |
| C | Até R$ 7.880 | 9,9% |
| B | Até R$ 15.760 | 2,8% |
| A | Acima de R$ 15.760 | 0,8% |

### Cor/Raça e Renda
- A cor **Amarela** apresenta a maior renda média
- A cor **Preta** apresenta a menor renda média

### Educação
- Existe uma relação positiva clara entre **anos de estudo e renda**
- Quanto maior a escolaridade, maior a renda média
